La guía básica de Git para el novato que llevas dentro.

Comandos basicos de la consola:

Estos comando son los que se utilizan reularmente en consola. Y no necesitan la palabra Git al inicio.

pwd
Muestra la ruta de la carpeta en la que te encuentres.
mkdir
Crear carpeta.
touch
Crear archivos.
ls
Lista de elementos dentro de un directorio.
ls -al
Muestra todos los archivos, hasta los ocultos (-a) en forma de lista (-l).
cd
Navegar entre carpetas.
cd ..
Regresar a carpeta anterior.
cd /
Ir a la ruta principal.
cd ~
Ir a la ruta del usuario.
history
Ver ultimos comandos ejecutados.
! + numero
Ejecuta el comando con el numero correspondiente de la lista de history.
